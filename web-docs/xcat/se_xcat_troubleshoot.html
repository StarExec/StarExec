xCAT Troubleshooting

Remove an execution host

In the example below we will remove n025.

Disable the node on the queue to avoid new jobs being allocated to the host
[root@se_m1 ~]# qmod -d all.q@n025

Remove the node from the cluster, command opens an editor, remove the lines referring to the host
[root@se_m1 ~]# qconf -mq all.q

Remove the node from allhosts group, this also opens an editor, remove lines referring to the host
[root@se_m1 ~]#qconf -mhgrp @allhosts

Remove the node from execution host list
[root@se_m1 ~]# qconf -de n025
